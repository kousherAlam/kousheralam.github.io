Attendance system with facial recognition aim to lower the friction in employee attendance. User can do their normal activity, they do not need to do anything. This attendance system will autometically recognize user and update their attendance.

## Technology Overview

This software consist of multiple layer of software and system intigrate as one single unit to provide high quality of user expeirence.
It use mqtt to communicate between ai and node backend. and use websocket between admin panel and the server. It provide realtime user attendance and the full monthly and yearly report
